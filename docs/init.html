Init

La aplicación UIF2 debe ser inicializada inmediatamente luego de incluido el script con el source.

//ejemplo de inicialización de la aplicacion
const uif2 = new Uif2({
  onLoaded: appLoaded
})

Parámetros de inicialización

[onLoaded]

Type: function

La function que se ejectuará luego de que la app se haya inicializado completamente
