Router

La aplicación permite utilizar un router para hacer llamadas asincrónicas, traer contenido remoto e insertarlo dentro de un contenedor en la página actual

//inicialización
const router = uif2.router({ container: "#targetContainer", cache: false });

//definición de rutas
router.add("routeNameA", "path/to/source", afterCallback, beforeCallback);
router.add("routeNameB", "path/to/source", afterCallback);
router.add("routeNameC", "path/to/source", afterCallback);

//correr una ruta
router.run("routeNameA");

Parámetros de inicialización

[container]

Type: string

Selector que indica el div contenedor en donde el router insertará el contenido remoto.

Métodos

add(name, pathToTheSource [,afterCallback] [,beforeCallback])

name Type: string

El nombre de la ruta. Con este nombre se podra identificar para luego ser llamada desde un link o desde el método run

//definición de ruta
router.add("routeNameA", "path/to/source");

<!-- link invocando la ruta -->
<a href="#routeNameA">mostrar ruta A</a>

pathToTheSource Type: string

El path desde donde el router hará un request para traer el contenido que insertará en el contenedor de destino.

afterCallback Type: string

Callback que se ejecutará luego de traer el contenido remoto e insertarlo en el DOM.

beforeCallback Type: string

Callback que se ejecutará antes traer el contenido remoto.
