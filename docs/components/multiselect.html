Multiselect

Crea un componente para selección multiple

<!-- se puede inicializar con parametros -->
<select class="uif-multiselect" data-source="/Documentation/MultiselectSource" data-id="Id" data-name="Empresa" multiple="multiple"></select>

// se puede inicializar por javascript
<select class="uif-multiselect" data-source="/Documentation/MultiselectSource" data-id="Id" data-name="Empresa" multiple="multiple"></select>

 $("#select5").UifMultiSelect({ sourceData: sourceArray });
 $("#select4").UifMultiSelect({ source: 'path/to/source' });

Parámetros

[data-sourceData]

Type: array

Array de valores que será el source del componente

[data-source]

Type: string

La url desde donde se obtendrá el array de valores

[data-id]

Type: string

El campo que se establecerá como ID

[data-name]

Type: string

El campo que se mostrará

[data-emptytext]

Type: string

El placeholder que se mostrará cuando no haya valores seleccionados

[data-numberdisplayed]

Type: number

La cantidad de registros visibles

[data-all]

Type: boolean

Permite agregar la opción de 'seleccionar todos'

[data-selectid]

Type: number

Permite preseleccionar por ID

Métodos

getSelected

Devuelve los valores seleccionados

const selectedItems = $('#multiselect').UifMultiSelect('getSelected');

setSelected

values

Type array

Selecciona opciones por su valor

$('#multiselect').UifMultiSelect('setSelected', [1]);

selectAll

Selecciona todos las opciones

$('#multiselect').UifMultiSelect('selectAll');

deSelectAll

Deselecciona todos las opciones

$('#multiselect').UifMultiSelect('deSelectAll');

deSelect

Deselecciona las opciones por su valor

$('#multiselect').UifMultiSelect('deSelect',[1]);

disabled

option

Type: boolean

Establece la propiedad disabled del componente

$('#multiselect').UifMultiSelect('disabled', true);

Eventos

  • binded
  • itemSelected
$('#multiselect').on('binded', function () {
    //component loaded
})

$('#multiselect').on('itemSelected', function (event, selectedItems) {
    //
})
