Datepicker

_El componente permite mostrar un input para seleccionar fechas

<!--primero se debe crear el elemento-->
<input type="text" value="13/01/2017" class="uif-datepicker" id="datepicker"  data-max-date="15/01/2017"/>

Parámetros

[data-min-date]

Type: string

Establece el valor mínimo para una fecha.

[data-max-date]

Type: string

Establece el valor máximo para una fecha.

Métodos

setValue

date

Type: string | date

Establece el valor

$("#datepicker").UifDatepicker('setValue', new Date());

$("#datepicker").UifDatepicker('setValue', '31/12/2016');

clear

Limpia el componente

$("#datepicker").UifDatepicker('clear');

getValue

Obtiene el valor ingresado en el componente

const value = $("#datepicker").UifDatepicker('getValue');

getHighlightedDates

Obtiene los días resaltados

const value = $("#datepicker").UifDatepicker('getHighlightedDates');

getDatesDisabled

Obtiene los días deshabilitados

const value = $("#datepicker").UifDatepicker('getHighlightedDates');

setDatesDisabled

dates

Type: string[]

Arreglo con las fechas a deshabilitar. Las fechas deben estar en formato DD/MM/YYYY

Configura los días deshabilitados Este método reconfigura completamente los días deshabilitados.

setHighlightedDates

dates

Type: string[] | []

Arreglo con las fechas a resaltar (en formato DD/MM/YYYY). Opcionalmente se puede pasar un arreglo de objetos con las siguientes propiedades:

date string Corresponde a la fecha (en formato DD/MM/YYYY)

class string El nombre de la clase CSS para aplicar a las fechas resaltadas

tooltip string Texto a mostrar en el tooltip de la fecha resaltada

Configura las fechas resaltadas.

const value = $("#datepicker").UifDatepicker('getValue');

disabled

option

Type: boolean

Establece la propiedad disabled del componente

$('#datepicker').UifDatepicker('disabled', true);

destroy

Elimina la funcionalidad del componente en el input

$('#datepicker').UifDatepicker('destroy');

Eventos

  • datepicker.change
  • datepicker.formatError
$('#datepicker').on("datepicker.change", function (event, date) {
    console.log(date);
});

//Evento se dispara si el usuario escribe un formato incorrecto en el control.
$('#datepicker').on('datepicker.formatError', function (event, value) {
    alert('Bad format : ' + value);
});
