Mask

El componente permite crear una máscara para darle formato a un input.

<!--se puede inicializar en el componente-->
<input type="text" class="uif-mask" data-pattern="000,00"  placeholder="___,__">

//se puede inicializar por JS
$('#example3').UifMask({ 
     pattern: '000000000000.00'
});


$('#example2').UifMask({
        pattern: '0ZZ.0ZZ.0ZZ.0ZZ', options: {
            translation: {
                'Z': {
                    pattern: /[0-9]/, optional: true
                }
            }
        }
    });

Parámetros del componente

[data-pattern]

Type: string

Establece el formato de la máscara

Parámetros

[pattern]

Type: string

Establece el formato de la máscara

Mas información:

Este componente es el único componente de framework que realiza un simple wrapping sobre un componente existente. El mismo esta basado en: https://igorescobar.github.io/jQuery-Mask-Plugin/

Por lo cual cualquier información adicional sobre el uso de mascaras puede verse en ese sitio.
