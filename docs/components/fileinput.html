Fileinput

El componente permite mostrar inputfile para adjuntar archivos


<input class="uif-inputfile" type="file" id="inputControlMultiple" data-input="false" data-text="Elija uno o varios archivos" multiple="multiple">

<input class="uif-inputfile"  type="file" id="inputControlSingle" data-input="true">

Parámetros

[multiple]

Type: string

multiple | ''

Especifica si el input acepta multiples archivos. El valor también puede estar vacío ej. <input class="uif-inputfile" multiple>

[data-text]

Type: string

El placeholder

[data-input]

Type: booleand

Habilita o dehabilita la entrada de texto
