FormatInput

El componente permitira formatear los input con el formato asignado.

<input class="uif-currency-format" type="text" />

Uso

En el archivo "uif-touch-4.0.0.es" se realiza una parametrizacion inicial por defecto, en el cual se define los valores con los cuales el formateo se va a realizar

$.uif2.defaults.decimalSeparator = ".";
$.uif2.defaults.thousandsSeparator = ",";
$.uif2.defaults.decimalPlaces = "3";

Este formato se puede ir modificando a lo largo de la aplicacion lo cual permite que todos los inputs modifiquen su comportamiento.
