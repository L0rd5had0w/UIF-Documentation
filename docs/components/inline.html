Inline

El componente permite crear un formulario con botones (prev, next, cancel, accept). Los botones disparan eventos. El componente puede ser usado para editar registros de una UifDatatable. --- Sólo debe utilzarse acompañado del componente UifDataTable en el caso que el mismo esté dentro de una ventana modal.

<form class="uif-inline" id="inlineForm">
    <div class="row">
        <div class="column">
            <label for="Empresa">Empresa</label>
            <input data-val="true" data-val-required="The Empresa field is required." id="Empresa" name="Empresa" type="text" value="" />
            <span class="field-validation-valid" data-valmsg-for="Empresa" data-valmsg-replace="true"></span>
        </div>
        <div class="column">
            <label for="Ciudad">Ciudad</label>
            <input data-val="true" data-val-required="The Ciudad field is required." id="Ciudad" name="Ciudad" type="text" value="" />
            <span class="field-validation-valid" data-valmsg-for="Ciudad" data-valmsg-replace="true"></span>
        </div>
        <div class="column">
            <label for="Pais">Pais</label>
            <input id="Pais" name="Pais" type="text" value="" />
        </div>
    </div>
</form>

Métodos

show

Oculta el form

$('#inlineForm').UifInline('show');

hide

Muestra el form

$('#inlineForm').UifInline('hide');
