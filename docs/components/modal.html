Modal

El componente permite mostrar un modal.

<div class="uif-modal" id="modalDialog" data-size="lg">
    <div class="uif-modalheader">       
    </div>

    <div class="uif-modalbody">
    </div>

    <div class="uif-modalbuttons">
    </div>
</div>

//parametro 1 : metodo
//parametro 2 : titulo del modal
//parametro 3(opcional) : enableClose Default : true
$('#modalDialog').UifModal('showLocal', 'Título del modal');

PArametros

size

<div class="uif-modal" id="modalDialog" data-size="lg">
    <div class="uif-modalheader">       
    </div>

    <div class="uif-modalbody">
    </div>

    <div class="uif-modalbuttons">
    </div>
</div>

Establece el ancho del modal

Type: string

Tamaños: xl | lg | md | sm

Default: lg

Métodos

showLocal

title

Type: string

Muestra el modal

//parametro 1 : metodo
//parametro 2 : titulo del modal
//parametro 3(opcional) : enableClose Default : true
$('#modalDialog').UifModal('show', 'Título del modal');

show

url

Type: string

title

Type: string

Muestra el modal con el contenido obtenido desde la url

//parametro 1 : metodo
//parametro 2 : url
//parametro 3 : titulo del modal
//parametro 4(opcional) : enableClose Default : true
$('#modalDialog').UifModal('show', 'path/to/modal', 'Título del modal');

changeTitle

title

Type: string

Cambia el titulo del modal url

$('#modalDialog').UifModal('changeTitle', 'Nuevo Titulo');

hide

Oculta el modal

$('#modalDialog').UifModal('hide');

Eventos

  • modal.opened
  • closed.modal

    //
});

$('#modal').on('closed.modal', function () {
    //
});
