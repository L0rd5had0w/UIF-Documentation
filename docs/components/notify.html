Notify

Muestra una notificación flotante

// notify sin autoclose
$.UifNotify('show', { type: 'info', message: 'Iniciando Proceso', autoclose: false });

// notify con autoclose
$.UifNotify('update', { type: 'success', message: 'Proceso Finalizado', autoclose: true });

Métodos

show

options Type: object

Muestra el notify. Ver opciones de configuración más abajo.

update

options Type: object Actualiza el notify. Ver opciones de configuración más abajo.

close

Cierra el notify

Parámetros del objeto options

type

Type: string

info | success | warning | danger

El estilo del notify.

message

Type: string

El texto del notify.

[autoclose]

Type: boolean

Especifica si el notify se cerrará automáticamente luego de 5 segundos.

[closeCallBack]

Type: function

Se ejecuta cuando se cierra la notificacion.

//Se ejecuta si se cierra la notificacion automaticamente o se cierra por el usuario.
var alert=function(){
    alert('La notificacion se cerro');
}
$.UifNotify('update', { type: 'success', message: 'Proceso Finalizado', autoclose: true,closeCallBack:alert });
