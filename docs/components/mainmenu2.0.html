Par치metros de inicializaci칩n

container

Type: string

El id del elemento que contendra el componente

//Ejemplo 
const container = '#myElement'

sourceData

Type: array

Data que se desea visualizar del menu

pathBase

Type: string

Url base del sitio

//Ejemplo 
const rootPath = '/SISE3G/'

searchCallBack

Type: function

Se ejecuta cuando un item del menu oprime opcion search

styleOptions

Type: object

Establece la configuracion de colores por defecto del menu.

//ejemplo de inicializaci칩n del componente
{
    primaryColor: '#04549c', //Color del menu que se encuentra en la parte derecha
    secondColor: '#4d83cb', //Color del menu cuando se abre un item del menu
    searchColor: '#d0e1ec', //Color del menu cuando los filtros estan activos
    titleColor: '#04549c', //Color de los titulos de los submenus.
    searchTitleColor: '#04549c', //Color de los titulos cuando el filtro esta activo.
    labelColorLeft: 'white', //Color de las fuentes de los menus padres de la parte izquierda
    labelColorRight: '#d0e1ec', //Color de las fuentes de los subitems
    labelColorSearch: '#414a4c', //Color de las fuentes cuando el filtro esta activo
}
