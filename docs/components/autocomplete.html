Autocomplete

El componente permite mostrar un mensaje dentro de un determinado elemento

<input class="uif-autocomplete" type="text" data-source="PATH_TO_THE_SOURCE" data-displaykey="Empresa" placeholder="the placeholder" data-displayrecords="5" id="inputtest">

$('#input').UifAutoComplete(
{
    source: url,
    displaykey="Empresa",
    minLenght: 3,
    queryParameter: 'query',
    displayRecords:5,
    errorSourceCallBack: function (status, error) {
        alert('Error load data Component.');
    },
    loadedCallback: function () {
        alert('Loaded Component.');
        $(this).validateSpecialCharacters();
    }
});

Parámetros

[data-source]

Type: string

Default: null

La url de donde se obtiene el listado de elementos

[data-displaykey]

Type: string

Default: null

El campo de el source que se usará para mostrar

[data-displayrecords]

Type: integer

Default: 5

El parametro define cuantos registros se van a mostrar en el control como maximo, sin importar los retornados por el Recurso, si el valor no es asignado por defecto es 5.

[data-minlenght]

Type: integer

Default: 3

El parametro define la cantidad minima de caracteres para inicializar la busqueda de registros.

[data-queryparameter]

Type: string

Default: query

El parametro define el nombre de la variable que sera enviada al servidor con el filtro.

[data-cleannoselected]

Type: boolean

Default: false

El parametro define, si no se selecciona un resultado de la busqueda, cuando el input pierda el focus limpiara el valor.

[loadedCallback]

Type: function

Default: null

Ejecuta la función cuando el componente carga completamente.

[errorSourceCallBack]

Type: function

Default: null

Ejecuta la función si la petición al servicio arroja error.

Métodos

setValue (value)

value

Type: string

Setea el valor

$('#inputtest').UifAutoComplete('setValue', 'Mirndia');

getValue ()

Obtiene el valor

$('#inputtest').UifAutoComplete('getValue');

clean ()

Limpia el control

$('#inputtest').UifAutoComplete('clean');

disabled (value)

value

Type: boolean

Habilita/Desabilita el control

$('#inputtest').UifAutoComplete('disabled', true);
