Asegurado Link

Este control permite un una configuracion rapida de los links para acceso a qsise.

<a id="linkAccess" class="btn btn-primary">Access QSISE</a>


$.fn.UifAseguradoLink.path = 'http://consultas.sistran.com.ar/consultas#/qsise/execute/';

$('#linkAccess').UifAseguradoLink('cb9c834d-e0ad-418e-8a95-cccad0e7af2d?IdAgente=1');

En la configuracion inicial se asigna en una variable global "$.fn.UifAseguradoLink.path" la cual asignara la ruta base de la aplicacion de qsise.

Usabilidad

$('#linkAccess').UifAseguradoLink('cb9c834d-e0ad-418e-8a95-cccad0e7af2d?IdAgente=1');

Permite asignar parametros al link, lo cual permitira el acceso a qsise con un parametro en especifico.
