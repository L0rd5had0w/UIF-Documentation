Dropdown

El componente permite hacer un dropdown con contenido dinámico

//ejemplo de inicialización del componente
const dropdown = app.dropDown({
                  source: 'modal-content.html',
                  element: '.open-dropdown',
                  align: 'left',
                  direction: 'top',
                  height: 300
                })

Parámetros de inicialización

source

Type: string

La url desde donde se va a obtener el HTML con el que se arma el dropDown

element

Type: string

Selector que contiene el elemento en donde se colocorá el dropDown

align

Type: string

left | right

El alineamiento horizontal con respecto al elemento de origen (target)

direction

Type: string

top | bottom

El alineamiento vertical con respecto al elemento de origen (target)

height

Type: number

El alto del dropdown

width

Type: number

El ancho del dropdown

loadedCallback

Type: function

Función de retorno que se ejecuta cuando el componente se ha cargado con los datos provistos por source.

Métodos

show

Muestra el dropdown

dropdown.show()
