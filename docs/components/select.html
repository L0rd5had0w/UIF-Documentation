Select

Crea un select con un source y eventos

<select class="uif-select" id="select1"></select>

$("#select1").UifSelect({ 
  sourceData: data, 
  id: "Id", 
  name: "Empresa", 
  native:false, 
  filter:true,
  enable:true,
  //displayTemplate: "mi nueva empresa es {{Empresa}}"
  //selectedId:15 --Asigna el value del select, despues de que carga.
  //size: 5 --Asigna las opciones a mostrar antes de realizar scroll, solo para select native:false
});

Parámetros de inicializacion

[sourceData]

Type: array

El array con los datos que serviran de source para el select.

[displayTemplate]

Type: string

Permite crear un template de visualizacion de los datos.

[id]

Type: string

El campo en el origen de datos que sera el ID.

[name]

Type: string

El campo en el origen de datos que será el name o el visible.

[displayTemplate]

Type: string

Este parametro permite personalizar el valor a mostrar, si este valor es llenado omitira el parametro [name]

[selectedId]

Type: object

Asigna el value del componente despues de que este haya cargado.

[filter]

Type: boolean

Indica si se usará el filtro.

[native]

Type: boolean

Indica si se usará el elemento estandar de HTML.

[enable]

Type: boolean

Indica si el componente esta habilitado o deshabilitado.

[size]

Type: int

Default: 15

Indica cantidad de registros a mostrar cuando los filtros estan activos.


//Cambio global.
$.fn.UifSelect.size = 15

[errorSourceCallBack]

Type: function

Default: null

Ejecuta la funcion si la peticion al servicio arroja error.

function callBackError (status, error) {
  //do something.
}
$("#select1").UifSelect({ 
  source: ul, 
  id: "Id", 
  name: "Empresa", 
  native:false, 
  filter:true,
  enable:true,
  errorSourceCallBack: callBackError
});

Métodos

disabled (value)

value

Type: boolean


$("#select").UifSelect("disabled", true);

setSelected(value)

value

Type: string

Selecciona un elemento buscando en el array de IDs que se estableció en la inicializacion.


$("#select").UifSelect("setSelected", "23365");

getSelected()

Retorna el ID del elemento seleccionado por el usuario.


const selectedID = $("#select").UifSelect("getSelected");

getSelectedText()

Retorna el name del elemento seleccionado por el usuario.


const selectedName = $("#select").UifSelect("getSelectedText");

getSelectedSource()

Retorna un objeto con el elemento seleccionado por el usuario.


const selected = $("#select").UifSelect("getSelectedSource");

getSource()

Retorna el origen de datos que tiene el select.


const source = $("#select").UifSelect("getSource");

setFocus()

Hace focus en el Select indicado


$("#select").UifSelect("setFocus");
