Alert

El componente permite mostrar un mensaje dentro de un determinado elemento

<!--primero se debe crear el elemento en donde se mostrará el mensaje-->
<div class="uif-alert" id="alert"></div>

//luego se puede invocar el método show
$("#alert").UifAlert('show', 'Bienvenido a UIF2 Touch!', "warning");

Métodos

show (message, type)

message

Type: string

type

Type: string

info | success | warning | danger

Muestra el alert con el formato especificado

$("#alert").UifAlert('show', 'Bienvenido a UIF2 Touch!', "success");

hide ()

Oculta el mensaje

$("#alert").UifAlert('hide');

appendMessage (message, cleanPreviousMessages)

Agrega un mensaje a mostrarse en el Alert. Los mensajes se mostraran en una lista.

message

Type: string

cleanPreviousMessages

Type: boolean Borrar los mensajes previos

$("#alert").UifAlert('appendMessage', 'Mensaje de ejemplo');

appendMessages (message, cleanPreviousMessages)

Muestra los mensajes en el Alert. Los mensajes se mostraran en una lista.

message

Type: string[]

cleanPreviousMessages

Type: boolean Borrar los mensajes previos

$("#alert").UifAlert('appendMessage', 'Mensaje de ejemplo');
