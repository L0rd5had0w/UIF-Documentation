Request

Helper para ejecutar requests asincronos. Retorna siempre una promesa

//inicialización
const request = new Uif2.request();

request.json('https://jsonplaceholder.typicode.com/posts/1')
  .then((response) => {
    console.log(response);
  })
  .catch((err) => {
    console.err(err);
  });

Métodos

json(url)

url Type: string

Ejecuta en GET sobre la url y devuelve un objeto.

request.json('https://jsonplaceholder.typicode.com/posts/1')
  .then((response) => {
    console.log(response);
  })

html(url)

url Type: string

Ejecuta en GET sobre la url y devuelve la respuesta como html.

request.html('https://jsonplaceholder.typicode.com/posts/1')
  .then((response) => {
    console.log(response);
  });

post(url)

url Type: string

Ejecuta en POST sobre la url.

const data = { name: 'asd', age: 34};
request.post('https://jsonplaceholder.typicode.com/posts/1', data)
  .then((response) => {
    console.log(response);
  });
