v4.2.0

Dropdown:
  • se cambió la propiedad 'element'. Ahora acepta solo un string con el selector. Ej. '#myButton'
Datepicker:
  • El método setValue solo acepta valores del tipo string o Date. Ej. .UifDatepicker('setValue', new Date())
  • Se agregó un método 'clear' para limpiar el control. Ej. .UifDatepicker('clear')
